# Code of Conduct

Onion Core aims to be a practical and respectful engineering project.

## Expected Behavior

- Be direct, specific, and respectful.
- Focus criticism on code, design, behavior, and evidence.
- Assume maintainers and contributors have limited time.
- Keep discussions relevant to the issue or pull request.
- Help make decisions easier by providing reproduction steps, examples, and
  tradeoffs.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or conduct.
- Publishing private information without permission.
- Repeated off-topic comments after maintainers ask to stop.
- Bad-faith reports or intentionally misleading reproductions.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; block accounts; or
limit participation when behavior harms the project.

For serious concerns, contact the maintainer privately or use GitHub moderation
and security reporting tools.
